# skycart-grocery
Modern online grocery shopping platform with real-time inventory, secure payments, order tracking, and fast delivery management.

## Features

- User authentication
- Product catalog
- Category management
- Shopping cart
- Wishlist
- Secure checkout
- Online payments
- Order tracking
- Delivery management
- Admin dashboard
- Inventory management
- Discounts and coupons
- Notifications

## Tech Stack

### Frontend
- javaScript
- React


### Backend
- java
- Springboot

### Database
- MySQL

### Authentication
- JWT
- OAuth

### Cloud
- Docker
- AWS / Azure / GCP
